# Architecture

## Stack
- Django (monolith)
- Django Templates
- Tailwind CSS
- PostgreSQL

## Apps
- core
- accounts
- weeks
- studies
- growth
- prayer
- journal
- announcements

## Core Rules
- Only one active week
- One check-in/takeaway/reflection per week per user
- DailyAlignment is unlimited
- Personal data is private

## Permissions
Leader:
- manages weekly content
- views user responses

Member:
- creates personal entries only
