# UI Direction

## Design Philosophy
- Calm
- Spacious
- Reflective
- Distraction-free

## Layout System
- Mobile-first vertical stacking
- Large spacing between sections
- Scroll-based

## Card System
All content lives inside cards

### Card Rules
- Rounded corners
- Soft backgrounds
- Generous padding
- Minimal shadow

### Card Types
- Primary (Weekly Word)
- Progress
- Info (Prayer, Gathering)
- Announcement

## Typography
- One primary font
- Clean hierarchy
- Minimal emphasis

## Colors
- Warm neutral background
- Dark text
- Soft accent tones

## Navigation
Bottom nav:
- Today
- Word
- Prayer
- Journal

## Interaction Rules
- No floating buttons
- No attention-grabbing animations
- No notifications

## UX Tone
- Quiet
- Intentional
- Unpressured
